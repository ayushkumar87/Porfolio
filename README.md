## 🌟 Modern React Portfolio – Built for Performance & Impact

A **premium, blazing-fast developer portfolio template** built with **React 18**, **Vite**, and **Vanilla CSS**, crafted to help developers showcase projects with **high performance, modern UI, and recruiter-focused design**.

🔗 **Live Demo:** https://ayush22205.vercel.app/

---

## ✨ Key Features (SEO Optimized)

* ⚡ **Lightning Fast Performance** – Powered by Vite for ultra-fast builds
* 🎨 **Modern UI/UX Design** – Dark theme with glassmorphism & gradients
* 🎬 **Interactive Animations** – Ripple effects, parallax, smooth transitions
* 🧠 **Centralized Data Management** – Update content from a single file
* 📱 **Fully Responsive Portfolio Website** – Mobile, tablet, desktop optimized
* 🧩 **Scalable React Architecture** – Easy to extend and customize
* 🔍 **SEO-Friendly Structure** – Optimized for discoverability & indexing

---

## 🛠️ Tech Stack

* **Frontend:** React 18
* **Build Tool:** Vite
* **Styling:** Vanilla CSS (CSS Variables)
* **Animations:** CSS Keyframes + Intersection Observer API
* **Icons:** SVG

## ⚡ Getting Started

### 1️⃣ Clone Repository

```bash
git clone https://github.com/Ayush-Kumar/portfolio-react.git
cd portfolio-react
```

### 2️⃣ Install Dependencies

```bash
npm install
```

### 3️⃣ Run Development Server

```bash
npm run dev
```

---

## 🎯 Easy Personalization

### 🔹 Update Portfolio Content

Edit:

```bash
src/data/portfolio.js
```

Update:

* 👤 Personal Info
* 🧠 Skills & Tech Stack
* 💼 Projects with Live Links
* 🎓 Education
* 📜 Certifications

---

### 🎨 Customize Theme

Edit CSS variables in:

```bash
src/App.css
```

Example:

```css
:root {
  --primary-color: #8b5cf6;
  --bg-primary: #0f172a;
  --text-primary: #f8fafc;
}
```

---

### 🧩 Add New Sections

1. Create new component in:

```bash
src/components/
```

2. Import in `App.jsx`

```jsx
import NewSection from "./components/NewSection";
```

3. Use inside JSX

```jsx
<NewSection />
```
